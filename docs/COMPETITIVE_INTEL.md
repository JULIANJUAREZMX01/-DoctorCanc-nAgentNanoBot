### 3.2 Patrones de Mensaje Real en WhatsApp (Formato Informal)

Basado en patrones observados en reseñas y foros, así escriben los clientes REALMENTE:

```
"Hola buenas tardes cuanto me sale cambiar la pantaya de un iphone 12"
"Oigan reparan samsun a54?"
"Se me cayo al agua mi cel que puedo hacer"

"Hola, mi cel no carga ¿pueden revisarlo?"
"Estan abiertos ahorita?"
"Donde estan ubicados?"
"Aceptan tarjeta?"

"tienen micas?"
"Hola! Se puede reparar un ipad?"

```

**Insights para el NLP del bot:**
- Abreviaciones comunes: "d" = de, "x" = por, "cel" = celular, "q" = que
- Faltas de ortografía frecuentes: "pantaya", "samsun", "huawey"
- Saludos: "Hola buenas tardes", "Oigan", "Buen dia"
- Emojis: Poco uso en preguntas, más en respuestas de agradecimiento
- El 80% de mensajes son de 1-2 líneas máximo

- El bot debe hablar en español coloquial mexicano (no formal)
- Las indicaciones de ubicación deben incluir referencias locales ("cerca de...", "por la Av. Puerto Juárez")
- No esperar alto volumen de turistas (zona residencial, no turística)
- Los clientes llegan caminando o en transporte local — priorizar rapidez

## 6. DIFERENCIADORES PARA EL BOT (vs Chatfuel/ManyChat)

### 6.1 Por Qué Propio y No SaaS

| Aspecto | Chatfuel/ManyChat | DoctorCancúnAgentNanoBot |
|---------|-------------------|-------------------------|
| Costo mensual | $15-$150 USD/mes | Costo de API LLM (~$5-20 USD) |
| Personalización | Templates limitados | 100% personalizable |
| IA conversacional | Básica / GPT genérico | LLM Router multi-proveedor con contexto |
| Datos del cliente | En servidor de tercero | Local / propio |
| Replicabilidad | Cero | Se clona para otro negocio en horas |
| Integración CRM | Plugins pagados | SQLite/Neon integrado |
| Branding | Logo de Chatfuel | Invisible, parece el dueño |
| Memoria | Limitada por sesión | Persistente (nanobot memory) |
| Cron/Follow-ups | Plan pagado | Incluido (nanobot cron) |
| Canales | FB Messenger + IG | WhatsApp + Telegram + Web |
